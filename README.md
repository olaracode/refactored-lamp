# Git

## Basic commands

- `git init` Initialize a repository
- `git add filename.txt` Add a file to the staging area
- `git add .` or `git add -A` Add all the files to the staging area
- `git commit -m "Your commit message"` Create a new commit(version) with the added files
- `git status` Check the current repository status

## Basic Workflow

- Make Changes to the desired(Any updates that you might need to do)

- **Stage the changes**: `git add -A`
- **Create the commit(new version)**: `git commit -m "Your descriptive commit message"`
- **Sync changes with github**: `git push`(if it's the first time you need to do `git push -u origin main`)
